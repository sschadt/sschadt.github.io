Project: Animal shelter intake analysis

Austin, TX animal shelter data analysis, including specific addresses of pet intakes. My role in this group project was to perform location-based analysis of pet intakes, to identify trends around:

  • Most frequent animal types by geographical area
  • Any trends around strays vs. animals in homes
  • Any correlation between intake frequency and proximity to veterinary clinics
Features: Google API polling, heat maps, map scatter plots

Tech stack:
  • Python
  • Pandas
  • Numpy
  • Jupyter Gmaps
  • Google Places API
  • Matplotlib
