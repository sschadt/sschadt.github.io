Project: Spotify music preferences by city

Analysis and visualization of Spotify listening preferences across 15 cities, displaying demographic data alongside top artists and song, clustering layers for genre, race, and gender filters.

Features: Web scraping, Spotify API polling, MongoDB database updates & retrieval, Flask API data endpoint, custom Leaflet.js/D3.js map layering

Tech stack:
  • Python
  • Spotipy API for Spotify
  • Flask / Flask-PyMongo
  • Leaflet.js
  • D3.js
  • Splinter
  • Beautiful Soup
  • Pandas
  • MongoDB
